固有値分解と信号の変換

固有値分解とは、行列を「固有値」と「固有ベクトル」という特別なパーツに分解する方法です。これを使うと、信号やデータを扱いやすくすることができます。特に、対称行列の場合は計算が簡単になり、直交行列を使って行列を対角化できます。

固有値分解の表現

行列 A を固有値分解すると、次のように書けます：
A = λ₁ q₁ q₁ᵀ + λ₂ q₂ q₂ᵀ + λ₃ q₃ q₃ᵀ + …

この式は、行列 A を固有値 λ₁, λ₂, … と固有ベクトル q₁, q₂, … の組み合わせで表していることを意味します。

ベクトル x を A で変換する場合、この表現を使うと次のように計算できます：
A x = λ₁ q₁ (q₁ᵀ x) + λ₂ q₂ (q₂ᵀ x) + …

ここで、qₖᵀ x はベクトル x を固有ベクトル qₖ に射影した結果を表します。

具体例

例えば、x = [ -1, 2, 3 ] を考えてみましょう：
x = 1 [1, 0, 0] + 2 [0, 1, 0] + 3 [0, 0, 1]

このベクトル x を行列 A で変換すると、次のようになります：
A x = Q Λ Qᵀ x = x

ここで、Q は固有ベクトルを並べた行列、Λ は固有値を対角成分にもつ行列です。

信号の畳み込みと変換

信号処理では、連続信号 x(t) とインパルス応答 h(t) の畳み込みを (x * h)(t) = ∫ x(τ) h(t - τ) dτ で定義します。
これをフーリエ変換を使って表すと、周波数成分同士の積として計算できます。

X(ω) H(ω) = ∫ x(t) e^(-jωt) dt × ∫ h(τ) e^(-jωτ) dτ

固有値分解と逆フーリエ変換

固有値分解は、逆フーリエ変換の考え方に似ています。行列 A の固有値分解を使うと、次のように信号を解析できます：

A = Σ (λₖ qₖ qₖᵀ) ⇒ A x = Σ (λₖ qₖ (qₖᵀ x))

この分解を使うことで、信号の特定の成分がどのように変換されるかを簡単に理解できます。
