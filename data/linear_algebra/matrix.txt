行列とは

行列は、数や式を表にしたものです。行列は、数学や物理学、工学などの様々な分野で利用されます。
行列は、以下のように表現されます。

A =
[ a₁₁ a₁₂ … a₁ₙ ]
[ a₂₁ a₂₂ … a₂ₙ ]
[ … … … … ]
[ aₘ₁ aₘ₂ … aₘₙ ]

基本的に大文字の英語で行列を表します。

用語の説明
行
行は、行列の横方向の要素を表します。先ほどの行列では、行は m 行あります。
列
列は、行列の縦方向の要素を表します。先ほどの行列では、列は n 列あります。
行列
行列は、数や式を表にしたものです。先ほどの行列は、m 行 n 列の行列といいます。
行列の大きさ
行列の大きさは、行数と列数を表します。先ほどの行列は、m × n の大きさの行列です。
要素
行列の要素 aᵢⱼ は、i 行 j 列に位置する数を表します。
正方行列
行列の行数と列数が等しい行列を正方行列といいます。正方行列は、n × n の大きさの行列です。
特殊な行列

単位行列
単位行列は、対角成分が1で、それ以外の成分が0の正方行列です。
単位行列は I か E で表すことが一般的です。

I =
[ 1 0 … 0 ]
[ 0 1 … 0 ]
[ … … … … ]
[ 0 0 … 1 ]

単位行列は、行列の掛け算において、単位元の役割を果たします。
簡単に言うと単位行列 I と任意の行列 A の積は、元の行列 A と等しくなります。

AI = IA = A

ゼロ行列
ゼロ行列は、全ての成分が0の行列です。
ゼロ行列は O で表すことが一般的です。

O =
[ 0 0 … 0 ]
[ 0 0 … 0 ]
[ … … … … ]
[ 0 0 … 0 ]
