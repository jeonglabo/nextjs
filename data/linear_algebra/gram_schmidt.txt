グラム・シュミットの直交化とは、ベクトル空間内の線形独立なベクトルの組を直交化する手法です。
直交化されたベクトルは、正規直交基底を構成します。

正規直交とは
正規とは、ベクトルの大きさが1であることを指します。ベクトル 𝒖 が ∥𝒖∥ = 1 の時、ベクトル 𝒖 は正規化されていると言います。
直交とは、2つのベクトルが直交することを指します。ベクトル 𝒖,𝒃 が直交する時、𝒖 ⟂ 𝒃 と表します。
また、直交するベクトルの内積は0になります。  
  𝒖 · 𝒃 = 0

直交化の手順
グラム・シュミットの直交化の手順は以下の通りです。以下は3つのベクトルの直交化の手順です。
ここで、𝒗︎₁,𝒗︎₂,𝒗︎₃ は線形独立なベクトルで、𝒖₁,𝒖₂,𝒖₃ は直交化されたベクトルです。

1. 𝒗︎₁ を正規化する。
   𝒖₁ = 𝒗︎₁ / ∥𝒗︎₁∥

2. 𝒗︎₂ に対して、𝒗︎₁ との直交成分を求め、正規化をする。
   𝒖₂ = (𝒗︎₂ - (𝒖₁ · 𝒗︎₂)𝒖₁) / ∥𝒗︎₂ - (𝒖₁ · 𝒗︎₂)𝒖₁∥

3. 𝒗︎₃ に対して、𝒗︎₁ と 𝒗︎₂ に直交する成分を求め、正規化をする。
   𝒖₃ = (𝒗︎₃ - (𝒖₁ · 𝒗︎₃)𝒖₁ - (𝒖₂ · 𝒗︎₃)𝒖₂) / ∥𝒗︎₃ - (𝒖₁ · 𝒗︎₃)𝒖₁ - (𝒖₂ · 𝒗︎₃)𝒖₂∥

具体例
𝒗︎₁,𝒗︎₂,𝒗︎₃ を以下のような3次元ベクトルとします。
  𝒗︎₁ = [1, 0, 1]ᵀ,  𝒗︎₂ = [1, 1, 0]ᵀ,  𝒗︎₃ = [1, 1, 1]ᵀ

1) 𝒗︎₁ を正規化して 𝒖₁ を求める:
   𝒖₁ = (1/√2)[1, 0, 1]ᵀ

2) 𝒗︎₂ に対して 𝒗︎₁ との直交成分を正規化して 𝒖₂ を求める:
   𝒖₂ = (𝒗︎₂ - (𝒖₁·𝒗︎₂)𝒖₁) / ∥𝒗︎₂ - (𝒖₁·𝒗︎₂)𝒖₁∥ = (1/√6)[1, 2, -1]ᵀ

3) 𝒗︎₃ に対して 𝒗︎₁ と 𝒗︎₂ との直交成分を正規化して 𝒖₃ を求める:
   𝒖₃ = (1/√3)[-1, 1, 1]ᵀ

最終的に、𝒖₁, 𝒖₂, 𝒖₃ はそれぞれ正規直交ベクトルになります。
